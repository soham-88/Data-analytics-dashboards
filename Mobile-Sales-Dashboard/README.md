# 📱 Mobile Sales Dashboard | Power BI Project

An interactive **Power BI Dashboard** built using mobile sales data to analyze revenue, transactions, quantity sold, customer ratings, payment methods, city-wise sales performance, and brand insights.

---

## 📌 Project Overview

This project transforms raw mobile sales data into a professional and interactive dashboard for:

- Sales Managers  
- Business Owners  
- Retail Analysts  
- Marketing Teams  
- Data Analysts  

The dashboard helps users monitor KPIs, track product performance, identify sales trends, and make data-driven business decisions.

---

## 🚀 Problem Statement

Mobile sales data is often stored in spreadsheets or raw reports, making it difficult to quickly analyze:

- Overall sales performance  
- Top-selling brands and models  
- Customer buying behavior  
- Preferred payment methods  
- City-wise revenue contribution  
- Monthly and daily sales trends  

This dashboard solves these challenges by providing a centralized analytics solution.
