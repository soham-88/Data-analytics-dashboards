# 📱 Mobile Sales Dashboard

## 📊 Overview
This project is an interactive Mobile Sales Dashboard built using Microsoft Power BI to analyze sales performance and generate business insights.

## 🛠️ Tools Used
- Microsoft Power BI
- Power Query
- DAX

## 🚀 Features
- KPI Tracking
- Sales Analysis
- Interactive Filters & Slicers
- Dynamic Visualizations

## 📈 Key Learnings
- Data Cleaning & Transformation
- Dashboard Design
- Business Insight Generation
