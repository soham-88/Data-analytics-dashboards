# 🎵 Spotify Top 50 World Dashboard

An interactive **Power BI Dashboard** built using Spotify's Top 50 World dataset to analyze songs, artists, rankings, popularity trends, album types, and music performance insights.

---

## 📌 Project Overview

This project transforms Spotify raw ranking data into a professional and interactive dashboard for:

- Music Analysts  
- Playlist Managers  
- Marketing Teams  
- Data Analysts  

The dashboard helps users monitor KPIs, compare artist performance, identify trending songs, and make data-driven decisions.

---

## 🚀 Problem Statement

Spotify Top 50 data is usually available in raw list format, making it difficult to quickly analyze:

- Song performance  
- Artist rankings  
- Popularity trends  
- Explicit vs Non-Explicit songs  
- Album type distribution  
- Monthly / Yearly growth trends  

This dashboard solves these challenges by providing a centralized analytics solution.

---

## 📊 Dashboard Pages

## 🔹 1. Overview Dashboard

Provides complete performance summary of Spotify Top 50 songs.

### Includes:

- Total Songs  
- Distinct Songs  
- Distinct Artists  
- Average Popularity  
- Average Duration  
- Explicit vs Non-Explicit Comparison  
- Album Type Breakdown  
- Songs by Year  
- Monthly Popularity Trends  
- Top Songs by Popularity  
- Top Artists by Popularity  

---

## 🔹 2. Artist Dashboard

Focused analysis of artist performance.

### Includes:

- Top Artists by Popularity  
- Songs by Artist  
- Tracks per Album  
- Avg Position Ranking  
- Artists with Most #1 Songs  
- Drill-down Artist Table  

---

## 🔹 3. Songs Dashboard

Detailed song-level insights.

### Includes:

- Top Songs Ranking  
- Song Count Comparison  
- Album / Single Distribution  
- Song Performance Table  
- Release Date Analysis  
- Duration Insights
