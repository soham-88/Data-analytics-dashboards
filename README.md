## 📌 About This Repository

This repository contains end-to-end Power BI dashboard projects built using real datasets.  
Each project covers the full analytics pipeline — from raw data import and cleaning in Power Query, to data modelling, DAX measure creation, and final interactive dashboard design.

These dashboards were built as part of my Data Science & Analytics learning at **Bhavan's College, Mumbai** and during my **Data Analytics Internship at Oasis Infobyte**.

---

## 🚀 Projects

### 📱 [Mobile Sales Dashboard](./Mobile-Sales-Dashboard/)

> Analysed mobile device sales performance across brands, cities, payment methods, and time periods.

- **Total Sales Tracked:** ₹769M across 4,000+ transactions
- **Brands Covered:** Apple, Samsung, Vivo, Xiaomi, OnePlus
- **Key Features:** KPI cards, city-wise map, brand comparison, monthly trends, payment insights
- **Tools:** Power BI · DAX · Power Query · Data Modelling

---

### 🎵 [Spotify Top 50 World Dashboard](./Spotify%20Top%2050%20World%20Analysis%20Dashboard/)

> Built a music analytics dashboard using the Spotify Top 50 World dataset to explore song popularity, artist trends, and album performance.

- **Dataset:** Spotify Top 50 World (global streaming chart data)
- **Key Features:** KPI cards, popularity trends, explicit content analysis, album type breakdown, artist ranking, DAX measures
- **Tools:** Power BI · DAX · Power Query · Data Modelling
